Halloween

Halloween is an e-commerce Web Application built for a seasonal Halloween shop. The user is able to view and book slots for schedules events, order and pay for decorations and confections. The user can also subscribe to the shop's newsletter for new products. The interface design is a result of collaboration between me and the client.

Coffee Shop

Coffee Shop is a simple blog website built for a local coffee shop. The user is able to view blogs by regular customers and coffee enthusiasts. Furthermore different types of beans and brews, pastas, and other pastries are also available for purchase. As usual, a newsletter is available for user subscription, offering the latest updates on the blogs, coffee and other available products.

Visit

Fratij Spares

Fratij Spares is an e-commerce and point of sale application built for a local vehicle spares shop. The application allows the user to manage their entire stock inventory, customers and suppliers. It also allows the users to create, view, update sales and track ongoing orders and product requests. Furthermore, the owner can go through daily statistics including sale accounts, supplier accounts and even revenue earnings for the day, week, month and compiled periodical statistics.

Lyriks

Lyriks is a music streaming application that allows users to play and listen to music from all over the world. It is a fully loaded application with every functionality you would require and enjoy on a musical platform, allowing the user to view the Top Charts from different regions, curate and share personal playlists and songs and much more.
